# Stock Monitor

## Set environment variables

### for produnction

1. edit `.env`
1. update environment variables value as below

```txt
VUE_APP_HOST=https://{YOUR FUNCTION APP HOST}/api
VUE_APP_KEY={YOUR API KEY}
```

### for local

1. create `.env.local`
1. add environment variables value as below

```txt
VUE_APP_HOST=https://{YOUR FUNCTION APP HOST}/api
VUE_APP_KEY={YOUR API KEY}
```
